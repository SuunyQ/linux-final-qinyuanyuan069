# parser项目 - Linux期末大作业
## 一、项目简介
本项目是Linux期末大作业核心模块，主要实现两大功能：
1. 字符串长度解析：接收任意字符串输入，返回其字符长度，支持空指针/空字符串异常处理；
2. 文件行数统计：读取指定文件，返回其总行数，支持文件不存在/无权限等异常场景处理。
项目包含完整的编译脚本（Makefile），可生成可执行文件、静态链接库、动态链接库，且支持Git版本控制，可托管至GitHub远程仓库。

## 二、使用环境
- 操作系统：Ubuntu/Debian/CentOS Linux（适配x86_64架构）
- 依赖工具：git（版本控制）、gcc（编译器）、make（编译脚本执行）、gdb（调试工具）

## 三、使用说明
### 1. 编译操作
```bash
# 进入项目目录
cd ~/parser

# 生成可执行文件（默认编译目标）
make

# 生成静态链接库libparser.a
make static

# 生成动态链接库libparser.so
make dynamic

# 清理所有编译产物（可执行文件/库文件/目标文件）
make clean
